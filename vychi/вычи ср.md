### Вариант 1

**1. Денормализованная форма числа с плавающей запятой**

Денормализованные (subnormal) числа были введены для решения проблемы «дырки» (underflow gap) в окрестности нуля и невозможности представить точный ноль в нормализованном виде.

Согласно стандарту IEEE 754, денормализованные числа рассчитываются по формуле:

$x_{subnorm} = -1^S \cdot M/2^n \cdot 2^{2-2^{w-1}}$ при смещенной экспоненте $E = 0$. В этом случае в старшем (скрытом) разряде мантиссы подразумевается $0$, а не $1$. При $E=0$ и $M=0$ получается $\pm 0$.

_Источник: Стр. 19-20, тема «Представление чисел в ЭВМ»._

**2. Метод простых итераций. Формула и поясняющий рисунок. Условие сходимости. Порядок метода**

- **Формула:** Исходное уравнение $f(x)=0$ приводится к виду $\phi(x) = x$. Итерационный процесс задается формулой $x_{n+1} = \phi(x_n)$. Если последовательность имеет предел, то $x^* = \lim_{n\to\infty} \phi(x_{n-1})$ является корнем.
    
- **Условие сходимости:** Метод сходится, если на интервале изоляции корня $[a, b]$ для производной выполняется условие $|\phi'(x)| \le q < 1$.
    
- **Порядок метода:** Первый порядок сходимости (погрешность уменьшается в геометрической прогрессии: $R_{n+1} \approx q R_n$).
    
- **Рисунок:** **Отсутствует в лекциях.** В тексте на стр. 41 есть заголовок «Пример расходящегося итерационного процесса», но самого графического поясняющего рисунка (паутинной диаграммы) для метода простых итераций в извлеченном тексте нет. _Пояснение от себя:_ Геометрически метод простых итераций представляет собой построение ломаной линии («паутинки» или «лесенки») между графиком функции $y = \phi(x)$ и прямой $y = x$, начиная от точки $x_0$, пока процесс не сойдется в точку их пересечения.
    
    _Источник: Стр. 39, 40, 43, тема «Численное решение уравнений»._
    

**3. Формула трапеций (простая и составная). Порядок точности**

- **Простая формула:** Заменяет функцию на полином Лагранжа первой степени. Интеграл оценивается как: $I \approx \frac{f(a) + f(b)}{2}[b - a]$. Порядок аппроксимации «простой» формулы — 3.
    
- **Составная (обобщенная) формула:** На равномерной сетке с шагом $h$: $I \approx h(\frac{1}{2}f_0 + f_1 + \dots + f_{n-1} + \frac{1}{2}f_n)$. Порядок аппроксимации составной формулы — 2 (алгебраический порядок — 1).
    
    _Источник: Стр. 98-99, тема «Численное интегрирование»._
    

**4. Формула интерполяционного многочлена Лагранжа**

Интерполяционный полином в форме Лагранжа ищется в виде $P_n(x) = \sum_{i=0}^n y_i \phi_i(x)$. Явная формула имеет вид:

$P_n(x) = \sum_{i=0}^n y_i \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}$.

_Источник: Стр. 66, тема «Интерполирование»._

**5. Что такое сходимость?**

- Для **разностных схем (дифференциальных уравнений):** Решение $y_n$, полученное с помощью схемы, сходится к точному решению $u(x_n)$ с порядком $p$, если $||y_n - u(x_n)|| \le C h^p$, где $C = const > 0$. (Теорема о сходимости: если схема аппроксимирует задачу и устойчива, то она сходится).
    
- Для **итерационных методов:** Итерационный метод сходится, если последовательность приближений $\{x^{(k)}\}$ имеет предел, равный точному решению $x^*$: $x^* = \lim_{k\to\infty} x^{(k)}$.
    
    _Источник: Стр. 173, 190, 308._
    

**6. Формула центральной разности для вычисления первой производной на равномерной сетке. Порядок аппроксимации (вывод)**

- **Формула:** $f'(x_i) \approx f_{\circ x,i} = \frac{f_{i+1} - f_{i-1}}{2h}$.
    
- **Вывод порядка аппроксимации: Отсутствует в лекциях.** (В материалах на стр. 119 приведен вывод только для _второй_ производной). _Вывод от себя:_ Разложим $f(x_{i+1})$ и $f(x_{i-1})$ в ряд Тейлора:
    
    $f(x_{i+1}) = f(x_i) + hf'(x_i) + \frac{h^2}{2}f''(x_i) + \frac{h^3}{6}f'''(x_i) + O(h^4)$
    
    $f(x_{i-1}) = f(x_i) - hf'(x_i) + \frac{h^2}{2}f''(x_i) - \frac{h^3}{6}f'''(x_i) + O(h^4)$
    
    Вычтем второе уравнение из первого:
    
    $f(x_{i+1}) - f(x_{i-1}) = 2hf'(x_i) + \frac{2h^3}{6}f'''(x_i) + O(h^5)$.
    
    Разделив на $2h$, получим: $f'(x_i) = \frac{f_{i+1} - f_{i-1}}{2h} - \frac{h^2}{6}f'''(x_i) + O(h^4)$. Отсюда видно, что погрешность пропорциональна $O(h^2)$, следовательно, порядок аппроксимации — второй.
    
    _Источник: Стр. 115, тема «Численное дифференцирование»._
    

**7. Формула неявного метода Эйлера, указать порядок метода — локальный и глобальный**

- **Формула:** $y_{n+1} = y_n + h f(x_{n+1}, y_{n+1})$.
    
- **Порядок метода: Отсутствует в лекциях.** (В лекциях на стр. 195 упоминается только свойство его абсолютной устойчивости, информация о порядке приведена только для явного метода Эйлера). _Ответ от себя:_ Как и явный метод Эйлера, неявный метод имеет локальный порядок аппроксимации 2 ($O(h^2)$) и глобальный порядок сходимости 1 ($O(h)$).
    
    _Источник: Стр. 195, тема «Численное решение ОДУ»._
    

**8. Какие численные методы для решения дифференциальных уравнений называются условно устойчивыми?**

Условно устойчивыми называют схемы (методы), где для обеспечения устойчивости численного решения на шаг интегрирования $h$ накладывается жесткое ограничение. (Например, в явной схеме Эйлера условие $|1 + h\lambda| < 1$ требует, чтобы $0 < h < 2/-\lambda$).

_Источник: Стр. 194, тема «Численное решение ОДУ»._

**9. Определение прямого дискретного преобразования Фурье. Какая у него минимальная асимптотическая сложность?**

- **Определение:** ДПФ — это линейное преобразование, задаваемое формальной формулой $\tilde{f}_j = C_+ \sum_{k=0}^{n-1} f_k \exp(2\pi i j k / n)$. (Также выражается через шаг временной сетки: $\tilde{f}_j = \frac{T}{n} \sum_{k=0}^{n-1} f_k \exp(i\omega_j t_k)$).
    
- **Сложность:** Прямое вычисление по определению требует $O(n^2)$ операций, но использование алгоритма Быстрого Преобразования Фурье (БПФ / FFT) снижает минимальную асимптотическую сложность до $O(N \log N)$.
    
    _Источник: Стр. 220-222, тема «Дискретное преобразование Фурье»._
    

**10. Метод прогонки. Идея алгоритма. Вычислительная сложность**

- **Идея алгоритма:** Метод представляет собой частный случай (вариант) метода Гаусса, адаптированный специально для решения СЛАУ с трехдиагональными матрицами (разреженными матрицами), получающимися, например, из трехточечных разностных уравнений. Метод состоит из прямого и обратного хода и применим при условии строгого диагонального преобладания.
    
- **Сложность:** Вычислительная сложность составляет $8n = O(n)$ арифметических операций.
    
    _Источник: Стр. 153-154, тема «Прямые методы решения СЛАУ»._

### Вариант 2

**1. Нормализованная форма числа с плавающей запятой**

В двоичном представлении нормализованное число $x$ записывается с помощью трех целых чисел (знака $S$, порядка $E$ и мантиссы $M$) по формуле:

$x = -1^S \cdot 1.M \cdot 2^E$.

Для исключения неоднозначности записи введено правило: первый разряд мантиссы нормализованного числа не равен нулю, а значит в двоичной записи он всегда равен единице ($1 \le M_b < b$), поэтому его можно явно не хранить в памяти.

_Источник: Стр. 16, тема «Представление чисел в ЭВМ»._

**2. Метод Ньютона. Формула и поясняющий рисунок. Порядок метода**

- **Формула:** Итерационный процесс (метод касательных) задается формулой $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$.
    
- **Порядок метода:** Для простого корня скорость сходимости метода Ньютона имеет порядок геометрической прогрессии (погрешность очередного приближения пропорциональна квадрату погрешности предыдущего: $x_n - x^* = \frac{1}{2}(x_{n-1} - x^*)^2 \phi''(\xi)$). То есть, метод имеет квадратичную сходимость.
    
- **Рисунок:** **Отсутствует в лекциях.** В тексте дано только словесное геометрическое описание. _Пояснение от себя:_ На графике функция заменяется её касательной в точке $(x_n, f(x_n))$. Новое приближение $x_{n+1}$ — это точка, где эта касательная пересекает ось абсцисс (ось $Ox$).
    
    _Источник: Стр. 45-46, тема «Численное решение уравнений»._
    

**3. Формула Симпсона (простая и составная). Порядок точности**

- **Простая формула:** Заменяет подынтегральную функцию на полином Лагранжа второй степени (формула парабол). $I \approx \frac{b-a}{6}[f(a) + 4f(\frac{a+b}{2}) + f(b)]$. Порядок аппроксимации «простой» формулы — 5.
    
- **Составная (обобщенная) формула:** Отрезок разбивается на четное число частей $N=2n$. $I \approx \frac{h}{3} \sum_{i=1,2}^{N-1} [f(x_{i-1}) + 4f(x_i) + f(x_{i+1})]$. Порядок аппроксимации составной формулы — 4 (алгебраический порядок — 3).
    
    _Источник: Стр. 100-101, тема «Численное интегрирование»._
    

**4. Формула интерполяционного многочлена Ньютона**

Многочлен ищется в виде полинома через разделенные разности. Формула Ньютона имеет вид:

$P_n(x) = y_0 + y(x_0, x_1)(x - x_0) + y(x_0, x_1, x_2)(x - x_0)(x - x_1) + \dots + y(x_0, x_1, \dots, x_n)(x - x_0)(x - x_1)\dots(x - x_{n-1})$.

_Источник: Стр. 70, тема «Интерполирование»._

**5. Что такое невязка (погрешность) аппроксимации?**

Невязка (погрешность) аппроксимации разностного уравнения ($\psi_n$ или $\psi_h$) — это результат подстановки _точного_ решения $u=u(x)$ в исходное разностное уравнение. В общем виде для разностной схемы $L_h u_h = f_h$ невязка определяется как $\psi_h = L_h (u)_h - f_h$.

_Источник: Стр. 173, 189, тема «Численное решение ОДУ»._

**6. Формула центральной разности для вычисления второй производной на равномерной сетке. Порядок аппроксимации (вывод)**

- **Формула:** $f''(x_i) \approx \frac{f_{i-1} - 2f_i + f_{i+1}}{h^2}$.
    
- **Порядок аппроксимации и вывод:** Порядок метода — $O(h^2)$. Вывод строится на разложении в ряд Тейлора-Маклорена. Складываем разложения для $y(x_i + h)$ и $y(x_i - h)$:
    
    $y_{i \pm 1} = y_i \pm h y'_i + \frac{h^2}{2}y''_i \pm \frac{h^3}{6}y'''_i + \frac{h^4}{24}y^{IV}(\eta_\pm)$.
    
    Подставляя их в исходную формулу, линейные и кубические члены взаимно уничтожаются, и мы получаем:
    
    $\frac{y_{i+1} - 2y_i + y_{i-1}}{h^2} = y''_i + \frac{h^2}{24}[y^{IV}(\eta_+) + y^{IV}(\eta_-)]$, откуда видно, что ошибка пропорциональна $h^2$.
    
    _Источник: Стр. 116, 119, тема «Численное дифференцирование»._
    

**7. Формула явного метода Эйлера, указать порядок метода — локальный и глобальный**

- **Формула:** Ограничиваясь первым членом разложения в ряд Тейлора, получаем схему ломаных: $y_{n+1} = y_n + h_n f(x_n, y_n)$.
    
- **Порядок метода:** Локальный порядок аппроксимации — 2. Глобальный порядок точности (сходимости) — 1 (т.е. $O(h)$).
    
    _Источник: Стр. 170, 172, тема «Численное решение ОДУ»._
    

**8. Какие численные методы для решения дифференциальных уравнений называются абсолютно устойчивыми?**

Абсолютно устойчивыми называются методы, которые при применении к модельному уравнению $y' = \lambda y$ (при $\Re\lambda < 0$) обеспечивают устойчивость ($y_n \to 0$ при $n \to \infty$) для _любого_ шага сетки $h$. Для таких методов область абсолютной устойчивости не накладывает жестких ограничений на выбор $h$ (например, как в неявной схеме Эйлера).

_Источник: Стр. 195, 196, тема «Численное решение ОДУ»._

**9. Определение обратного дискретного преобразования Фурье. Какая у него минимальная асимптотическая сложность?**

- **Определение:** Обратное ДПФ — это линейное преобразование, восстанавливающее функцию по её спектру. Формула: $f_k = C_- \sum_{j=0}^{n-1} \tilde{f}_j \exp(-2\pi i j k / n)$ (или через временную сетку: $f_k = \frac{1}{T} \sum_j \tilde{f}_j \exp(-i\omega_j t_k)$).
    
- **Сложность:** Прямое вычисление требует $O(n^2)$ операций, однако алгоритм БПФ сводит минимальную асимптотическую сложность к $O(N \log N)$.
    
    _Источник: Стр. 220-222, тема «Дискретное преобразование Фурье»._
    

**10. Метод Гаусса. Идея алгоритма, варианты метода Гаусса. Вычислительная сложность**

- **Идея:** Алгоритм прямого метода решения СЛАУ, основанный на матричном LU-разложении ($A=LU$). В прямом ходе основная матрица системы приводится к ступенчатому (верхнему треугольному) виду с помощью последовательного зануления поддиагональных элементов. В обратном ходе неизвестные вычисляются последовательной подстановкой.
    
- **Варианты метода:** Чтобы избежать деления на ноль или повысить устойчивость алгоритма, используется метод Гаусса с выбором ведущего элемента (перестановка строк/столбцов): с _частичным_ выбором ведущего элемента по столбцу и с _полным_ выбором ведущего элемента (по всей матрице).
    
- **Сложность:** Прямой ход требует $\approx \frac{2}{3}n^3 = O(n^3)$ арифметических операций. Обратный ход требует $O(n^2)$ операций. Итоговая вычислительная сложность — $O(n^3)$.
    
    _Источник: Стр. 144, 145, 148, 149, тема «Прямые методы решения СЛАУ»._

   * Вариант 1: Сходимость. Свойство численного метода, при котором его решение стремится к точному решению исходной задачи при уменьшении шага

  6. Численное дифференцирование (стр. 115, 116, 119)

  9. Дискретное преобразование Фурье (ДПФ) (стр. 221, 222)

# Дополнительные темы и определения с формулами

### 1. Итерационные методы решения СЛАУ
* **Что делают:** Находят приближенное решение системы алгебраических уравнений $Ax=b$ путем последовательного уточнения начального вектора. Позволяют эффективно решать огромные разреженные системы.
* **Метод Якоби:** Итерационно обновляет все неизвестные, используя значения только с предыдущего шага. 
    * *Формула:* $x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right)$
    * *Условие сходимости:* Строгое диагональное преобладание: $|a_{ii}| > \sum_{j \neq i} |a_{ij}|$.
* **Метод Зейделя:** Модификация Якоби, где для обновления текущей неизвестной используются уже найденные («свежие») значения на текущем шаге. Обычно сходится быстрее.
    * *Формула:* $x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j < i} a_{ij} x_j^{(k+1)} - \sum_{j > i} a_{ij} x_j^{(k)} \right)$
* **Метод Сопряженных градиентов (CG):** Решает системы с симметричными положительно определенными матрицами. Ищет решение как точку минимума квадратичного функционала, двигаясь по оптимальным (A-ортогональным) направлениям. В точной арифметике сходится за конечное число шагов (не более $n$).
    * *Минимизируемый функционал:* $\Phi(x) = \frac{1}{2}(Ax, x) - (b, x)$
    * *Итерация:* $x_{k+1} = x_k + \alpha_k p_k$, где $p_k$ — сопряженные направления ($(Ap_i, p_j) = 0$).

### 2. Приближение функций и Сплайны
* **Что делают:** Восстанавливают функцию по дискретному набору точек (для интерполяции или сглаживания данных).
* **Кубический сплайн:** Строит гладкую кривую, проходящую точно через заданные узлы. На каждом отрезке функция представляется полиномом 3-й степени, при этом на стыках сохраняется непрерывность самой функции, а также ее 1-й и 2-й производных.
    * *Вид на отрезке $[x_i, x_{i+1}]$:* $S_i(x) = a_i + b_i(x-x_i) + \frac{c_i}{2}(x-x_i)^2 + \frac{d_i}{6}(x-x_i)^3$
* **Метод наименьших квадратов (МНК):** Строит аппроксимирующую функцию (например, прямую или параболу), которая не обязана проходить точно через узлы, но минимизирует общую ошибку. Идеально для «шумных» экспериментальных данных.
    * *Целевая функция:* $F(a_0, \dots, a_m) = \sum_{i=1}^N (P_m(x_i) - y_i)^2 \to \min$
    * *Нормальная система (для прямой $y = ax + b$):* 
      $\begin{cases} a\sum x_i^2 + b\sum x_i = \sum x_i y_i \\ a\sum x_i + b \cdot n = \sum y_i \end{cases}$

### 3. Проблема собственных значений
* **Что делают:** Численно находят собственные значения (спектр) и собственные векторы матриц.
* **Степенной метод:** Решает «частичную» проблему — находит только одно, **наибольшее по модулю** собственное значение и его вектор. Метод заключается в многократном умножении произвольного начального вектора на матрицу.
    * *Итерация:* $v^{(k+1)} = \frac{Av^{(k)}}{\|Av^{(k)}\|}$
    * *Приближение собственного значения:* $\lambda_1 \approx \frac{(Av^{(k)}, v^{(k)})}{(v^{(k)}, v^{(k)})}$ (отношение Рэлея).
* **QR-алгоритм:** Решает «полную» проблему — находит все собственные значения матрицы, итеративно приводя её к верхней треугольной (или блочно-треугольной) форме с помощью ортогональных QR-разложений.

### 4. Краевые задачи для ОДУ
* **Что делают:** Находят решение дифференциального уравнения на отрезке, если граничные условия заданы на обоих концах этого отрезка (в отличие от задачи Коши, где всё задано в одной начальной точке).
* **Метод стрельбы:** Сводит краевую задачу к серии задач Коши. Мы задаем произвольное недостающее начальное условие на одном конце («прицеливаемся»), решаем как задачу Коши и смотрим, попали ли в заданное условие на другом конце. Если нет — корректируем начальное значение (например, методом Ньютона).
* **Метод конечных разностей (сеточный):** Заменяет дифференциальное уравнение системой алгебраических уравнений на дискретной сетке.
    * *Порядок аппроксимации:* Замена центральными разностями дает $O(h^2)$ (второй порядок).
    * *Замена производных для $u'' + p(x)u' + q(x)u = f(x)$:*
      $\frac{u_{i+1} - 2u_i + u_{i-1}}{h^2} + p_i \frac{u_{i+1} - u_{i-1}}{2h} + q_i u_i = f_i$
    * Полученная СЛАУ имеет трехдиагональную матрицу и эффективно решается **методом прогонки**.

### 5. Уравнения в частных производных (УЧП)
* **Что делают:** Решают дифференциальные уравнения, зависящие от нескольких переменных (например, от времени $t$ и координаты $x$), заменяя непрерывную область на дискретную сетку (метод сеток).
* **Уравнение теплопроводности (параболическое):** $u_t = a u_{xx}$. Описывает процессы выравнивания (диффузия, тепло).
    * *Явная схема:* Вычисляет новый слой по времени явно через предыдущий. Порядок аппроксимации: $O(\tau + h^2)$ (1-й по времени, 2-й по пространству).
    * *Формула:* $\frac{u_i^{n+1} - u_i^n}{\tau} = a \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{h^2}$
    * *Условие устойчивости (КФЛ):* Жесткое ограничение на шаг: $\sigma = \frac{a\tau}{h^2} \le \frac{1}{2}$
* **Волновое уравнение (гиперболическое):** $u_{tt} = c^2 u_{xx}$. Описывает распространение колебаний.
    * *Схема «Крест»:* Явная схема. Порядок аппроксимации: $O(\tau^2 + h^2)$ (2-й по времени и по пространству).
    * *Формула:* $\frac{u_i^{n+1} - 2u_i^n + u_i^{n-1}}{\tau^2} = c^2 \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{h^2}$
    * *Условие устойчивости:* Число Куранта $c\frac{\tau}{h} \le 1$.
* **Уравнение Пуассона (эллиптическое):** $u_{xx} + u_{yy} = f(x, y)$. Описывает стационарные (не зависящие от времени) поля.
    * *Схема «пятиточечный шаблон»:* Заменяет вторые производные по осям X и Y разностями. Порядок аппроксимации: $O(h_x^2 + h_y^2)$.
    * *Формула:* $\frac{u_{i+1,j} - 2u_{i,j} + u_{i-1,j}}{h_x^2} + \frac{u_{i,j+1} - 2u_{i,j} + u_{i,j-1}}{h_y^2} = f_{i,j}$
