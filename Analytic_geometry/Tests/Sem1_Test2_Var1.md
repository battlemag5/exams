# Аналитическая геометрия: Контрольная работа 2. Вариант 1

### Задача 1
Даны три прямые:

$$
\cases{ x = 3 + t, \cr y = -1 + 2t, \cr z = 4t. } \qquad 
\cases{ x = -2 + 3t, \cr y = -1, \cr z = 4 - t. } \qquad 
\cases{ x - 3y + z = 0, \cr x + y - z + 4 = 0. }
$$

Написать уравнение прямой, пересекающей две из данных прямых и параллельной третьей прямой.  
*Система координат аффинная.*

### Задача 2
Установить взаимное расположение следующих прямых:

$$
\cases{ x + z - 1 = 0, \cr 3x + y - z + 13 = 0. } \qquad 
\cases{ x - 2y + 3 = 0, \cr y + 2z - 8 = 0. }
$$

Если они принадлежат одной плоскости — выписать уравнение этой плоскости. Если прямые скрещиваются — выписать уравнение плоскости, проходящей через первую прямую параллельно второй.  
*Система координат аффинная.*

### Задача 3
Написать уравнение общего перпендикуляра к двум прямым:

$$
\frac{x - 1}{8} = \frac{y - 2}{4} = \frac{z - 3}{1} \qquad \frac{x - 1}{1} = \frac{y}{1} = \frac{z}{1}
$$

Найдите расстояния между этими прямыми. Найти точки пересечения общего перпендикуляра к данным прямым с этими прямыми.  
*Система координат декартова.*

### Задача 4
Написать уравнение плоскости, проходящей через прямую

$$
\frac{x + 7}{-2} = \frac{y - 6}{3} = \frac{z}{1}
$$

и образующей угол $\frac{\pi}{3}$ с прямой $x - y + z = 0, \ x - y + 2z = 0$.

### Задача 5
Найти расстояние от точки $(1, 3, 5)$ до прямой

$$
\cases{ 2x + y + z - 1 = 0, \cr 3x + y + 2z - 3 = 0. }
$$

*Система координат декартова.*
