# Математический анализ. Семестр 4. Контрольная работа 1. Вариант 2.

Пусть 

$$
F(\alpha) = \int_{0}^{\infty} \frac{e^{-\alpha \sqrt{x}} \cos x}{\sqrt[4]{x^3}} \ dx
$$

1. Найдите все значения параметра $\alpha$, при которых интеграл сходится.

2. Докажите, что функция $F(\alpha)$ непрерывна по $\alpha$ в области сходимости интеграла.

3. Докажите, что функция $F(\alpha)$ непрерывно дифференцируема по $\alpha$ внутри области сходимости интеграла.

4. Найдите $G'(\alpha)$, если:

$$
G(\alpha) = \int_{e^{-\alpha}}^{e^{\alpha}} \frac{\ln(1 + \alpha^2 x^2)}{x} \ dx
$$

5. Выразите интеграл ($0 < p < 2$) через эйлеровы интегралы:

$$
\int_{0}^{+\infty} \frac{x^{p-1}}{(1 + x)^2} \ dx
$$

6. Применяя полученный результат, вычислите интеграл:

$$
\int_{0}^{\infty} \frac{\sqrt{x} \ln x}{(1 + x)^2} \ dx
$$
