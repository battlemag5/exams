# Семестр 1. Контрольная работа 1. Вариант 1.

1. Определите, к какому соотношению ($X \subset Y, Y \subset X, X = Y$) находятся множества $X$ и $Y$, если
$$X = A \setminus (B \cup C), Y = (A \setminus B) \cup (A \setminus C).$$
Ответ обоснуйте.

2. Функция $f : \mathbb{R} \to \mathbb{R}$ задана как $f(x) = x[x]$, где $[x]$ - целая часть числа $x$. Множество $A = [-2; 2]$. Найдите $f(A), f^{-1}(A), f(f^{-1}(A)), f^{-1}(f(A))$.

3. Применяя метод математической индукции, докажите равенство
$$\sum_{k=1}^{n} \sin(2k - 1)x = \frac{\sin^2(nx)}{\sin x}, x \neq 2\pi m.$$

4. Применяя метод математической индукции, докажите, что для любых положительных чисел $a_1, a_2, ..., a_n$ справедливо неравенство
$$(a_1 + a_2 + ... + a_n)\left(\frac{1}{a_1} + \frac{1}{a_2} + ... + \frac{1}{a_n}\right) \ge n^2.$$

5. Найдите точную верхнюю и нижнюю границу множеств
$$A = \left\{ \frac{2n^2 - 1}{2 + n^2}, n \in \mathbb{N} \right\} \text{ и } B = \{(1 - n)^{\sin(\pi n / 2)}, n \in \mathbb{N}\}.$$
Существуют ли у этих множеств наибольший и наименьший элементы? Если существуют, найдите их.
