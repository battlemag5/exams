2. Выясните, какое из множеств $A$ или $B$ является гладким многообразием без края:

$$
A = \{ (x,y,z) \in \mathbb{R}^3 : y^2 + xy + xz + 1 = 0, \ 2xz + y + 2 = 0 \}
$$

$$
B = \{ (x,y,z) \in \mathbb{R}^3 : y^2 + xy + xz + 1 = 0, \ 2xz - y = 0 \}
$$

Укажите размерность этого многообразия, а также размерности касательного и нормального пространств к нему. Найдите уравнения касательного и нормального пространств к нему в точке $(2, -2, -1/2)$.

Приведите пример дополнительного условия, с помощью которого можно превратить множество из этой задачи, не являющееся многообразием, в гладкое многообразие.
