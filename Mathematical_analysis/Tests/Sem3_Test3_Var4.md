## Контрольная работа 3. Вариант 4.

1. Измените порядок интегрирования:
$$\int_{\pi/2}^{3\pi/4} dx \int_{\operatorname{ctg} x}^{0} f(x, y) \, dy + \int_{3\pi/4}^{\pi} dx \int_{\operatorname{tg} x}^{0} f(x, y) \, dy$$

2. Используя полярную замену переменных, вычислите интеграл по области $D = \{(x, y) \in \mathbb{R}^2, 4y < x^2 + y^2 < 6y, x < y\}$:
$$\iint_{D} x \, dxdy$$

3. Используя подходящую замену переменных, найдите площадь плоской фигуры, ограниченной линиями:
$$y = x^2, \quad y = 16x^2, \quad xy^2 = 1, \quad xy^2 = 4.$$

4. Вычислите интеграл:
$$\iiint_{G} z \, dxdydz.$$
$G$ — область в $\mathbb{R}^3$, ограниченная поверхностями
$$x^2 + y^2 = 1, \quad z = 1, \quad z = x + y, \quad x > 0, \quad y > 0.$$
