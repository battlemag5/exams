# Математический анализ. Семестр 3.
## Контрольная работа 3. Вариант 2.

1. Измените порядок интегрирования:
$$\int_{-\pi/2}^{0} dx \int_{\sin x}^{\cos x} f(x, y) \, dy$$

2. Используя полярную замену переменных, вычислите интеграл по области $D = \{(x, y) \in \mathbb{R}^2, 1 \le x^2 + y^2 \le 2y\}$:
$$\iint_{D} \frac{y}{\sqrt{x^2 + y^2}} \, dxdy$$

3. Используя подходящую замену переменных, найдите площадь плоской фигуры, ограниченной линиями:
$$y = x^4, \quad y = 8x^4, \quad xy^2 = 1, \quad xy^2 = 8.$$

4. Вычислите интеграл:
$$\iiint_{G} z^2 \, dxdydz.$$
$G$ — область в $\mathbb{R}^3$, ограниченная плоскостями
$$x + y = 0, \quad x - z = 0, \quad y - z = 0, \quad z = 1.$$
