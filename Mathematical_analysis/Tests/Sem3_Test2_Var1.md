$$

2. Выясните, какое из множеств $A$ или $B$ является гладким многообразием без края:

$$
A = \{ (x,y,z) \in \mathbb{R}^3 : z^2 + xy - xz + 2 = 0, \ xy - z + 2 = 0 \}
$$

$$
B = \{ (x,y,z) \in \mathbb{R}^3 : z^2 + xy - xz + 2 = 0, \ xy + z - 2 = 0 \}
$$

Укажите размерность этого многообразия, а также размерности касательного и нормального пространств к нему. Найдите уравнения касательного и нормального пространств к нему в точке $(5, 0, 2)$. 

Приведите пример дополнительного условия, с помощью которого можно превратить множество из этой задачи, не являющееся многообразием, в гладкое многообразие.

3. Найдите условные экстремумы функции $f(x,y,z) = x^2 + z^2$ на многообразии $M$:

$$
M = \{ (x,y,z) \in \mathbb{R}^3 : x^2 + y^2 = 1, \ y + z = 2 \}
$$
