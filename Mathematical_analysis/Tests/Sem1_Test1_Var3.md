# Семестр 1. Контрольная работа 1. Вариант 3.

1. Определите, к какому соотношению ($X \subset Y$, $Y \subset X$, $X = Y$) находятся множества $X$ и $Y$, если:
<p align="center">$X = A \cup (B \setminus C), \quad Y = (A \cup B) \setminus (A \cup C)$</p>
Ответ обоснуйте.

2. Функция $f : \mathbb{R} \to \mathbb{R}$ задана как $f(x) = x + [x]$, где $[x]$ — целая часть числа $x$. Множество $A = [-2; 2]$. Найдите $f(A)$, $f^{-1}(A)$, $f(f^{-1}(A))$, $f^{-1}(f(A))$.

3. Применяя метод математической индукции, докажите равенство:
<p align="center">$\sum_{k=0}^{n} \cos 2kx = \frac{\sin((n + 1)x) \cos(nx)}{\sin x}, \quad x \neq 2\pi m$</p>

4. Применяя метод математической индукции, докажите, что для любых действительных чисел $a_1, a_2, ..., a_n$ справедливо неравенство:
<p align="center">$(a_1 + a_2 + ... + a_n)^2 \le n(a_1^2 + a_2^2 + ... + a_n^2)$</p>

5. Найдите точную верхнюю и нижнюю границу множеств:
<p align="center">$A = \left\{ \frac{n + (-1)^n}{3n - 1}, \, n \in \mathbb{N} \right\}$</p>
и
<p align="center">$B = \left\{ (-n)^{(-1)^n}, \, n \in \mathbb{N} \right\}$</p>
Существуют ли у этих множеств наибольший и наименьший элементы? Если существуют, найдите их.
